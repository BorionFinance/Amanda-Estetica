# Constelação · Borion System

Aplicativo-hub instalável do ecossistema Borion.

## Instalação
- **Computador:** abra o site publicado e clique em **Baixar App**.
- **Celular:** abra o site no navegador compatível e toque em **Baixar App** ou use a opção **Adicionar à tela inicial / Instalar aplicativo** do navegador.

## Conteúdo
- `index.html` — página principal do hub.
- `manifest.webmanifest` — manifesto PWA.
- `sw.js` — service worker.
- `icon-192.png`, `icon-512.png`, `apple-touch-icon.png`, `favicon-32.png`, `favicon.ico` — ícones oficiais do app.
- `icon-source-1254.png` — arte principal em alta resolução.
- `assets/icons/` — variações em PNG geradas para o projeto.

## Nome do app
O PWA foi configurado para ser instalado com o nome **Constelação**.
